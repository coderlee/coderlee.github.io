CSS Length Units Reference    长度单位

 
Length Units
长度单位		CSS Version
版本		Compatibility
兼容性		Description
简介
相对长度单位  Relative Length Units
emCSS1IE4+ , NS4+相对于当前对象内文本的字体尺寸
exCSS1IE4+ , NS4+相对于字符 “ x ” 的高度。通常为字体高度的一半
pxCSS1IE3+ , NS4+像素（Pixel）
绝对长度单位  Absolute Length Units
ptCSS1IE3+ , NS4+点（Point）
pcCSS1IE3+ , NS4+派卡（Pica）。相当于我国新四号铅字的尺寸
inCSS1IE3+ , NS4+英寸（Inch）
cmCSS1IE3+ , NS4+厘米（Centimeter）
mmCSS1IE3+ , NS4+毫米（Millimeter）

 
单位换算：1in = 2.54cm = 25.4 mm = 72pt = 6pc
 
 
说明：这种颜色是CSS2标准单位。这种颜色是IE建议样式表单位。这种颜色是NS私有单位。这种颜色是目前尚无浏览器支持的单位。
