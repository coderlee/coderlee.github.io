pitch-range版本：CSS2　 兼容性：NONE　继承性：有

 
语法：
 
pitch-range : number
 
参数：
 
number 数字。取值范围为0-100。默认值为50，表示普通发音。小于50趋向于平缓，大于50趋向于颤抖。
 
说明：
 
设置或检索声音的平滑程度。
目前IE5.5尚不支持此属性。
对应的脚本特性为pitchRange。请参阅我编写的其他书目。
 
示例：
 
strong { pitch-range: 80 }
