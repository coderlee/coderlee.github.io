CSS Angle Units Reference    角度单位

 
Angle Units
角度单位		CSS Version
版本		Compatibility
兼容性		Description
简介
degCSS2NONE度。一个圆圈的360等分之一
gradCSS2NONE梯度。一个直角的100等分之一。一个圆圈相当于400grad
radCSS2NONE弧度。把一个圆圈分成2*PI单位

 
 
说明：这种颜色是CSS2标准单位。这种颜色是IE建议样式表单位。这种颜色是NS私有单位。这种颜色是目前尚无浏览器支持的单位。
