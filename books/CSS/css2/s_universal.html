Universal Selector    通配选择符

 
语法：
 
*
 
说明：
 
选定文档目录树（DOM）中的所有类型的单一对象。
假如通配选择符不是单一选择符中的唯一组成，“*”可以省略。
目前IE5.5+尚不支持此种选择符。
 
示例：
 
*[lang=fr] { font-size:14px; width:120px; }
*.div { text-decoration:none; }
