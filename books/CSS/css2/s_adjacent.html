Adjacent Sibling Selectors    相邻选择符

 
语法：
 
E1 + E2
 
说明：
 
选择紧贴在对象E1之后的所有E2对象。E2和E1对象在文档目录结构树（DOM）中有共同的父对象。
目前IE5.5+尚不支持此种选择符。
 
示例：
 
div + p { font-size:14px; }
/* 所有紧贴在div对象之后的p对象的字体尺寸为14px */
 
div.fly + p { font-size:14px; }
