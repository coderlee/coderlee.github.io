cue-before版本：CSS2　 兼容性：NONE　继承性：无

 
语法：
 
cue-before : none | url (url)
 
参数：
 
none : 　无声音播放
url : 　使用绝对或相对地址指定音乐文件
 
说明：
 
设置在对象前播放的声音。
目前IE5.5尚不支持此属性。
对应的脚本特性为cueBefore。请参阅我编写的其他书目。
 
示例：
 
div { cue-before: url(orchestralsneeze.wav) ; }
