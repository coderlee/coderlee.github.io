:first-child版本：CSS2　 兼容性：NONE

 
语法：
 
Selector1 Selector2 : first-child { sRules }
 
说明：
 
设置对象（Selector1）的第一个子对象（Selector2）的样式表属性。
目前IE5.5尚不支持此伪类。
 
示例：
 
p a:first-child { color: green }
table td:first-child { width:200px; }
