word-wrap版本：IE5.5+专有属性　继承性：有

 
语法：
 
word-wrap : normal | break-word
 
参数：
 
normal : 　允许内容顶开指定的容器边界
break-word : 　内容将在边界内换行。如果需要，词内换行（word-break）也行发生
 
说明：
 
设置或检索当当前行超过指定容器的边界时是否断开转行。
对应的脚本特性为wordWrap。请参阅我编写的其他书目。
 
示例：
 
div { word-wrap: break-word; }
