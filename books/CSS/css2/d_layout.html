CSS Layout Properties    布局属性

 
Layout Properties
属性		CSS Version
版本		Compatibility
兼容性		Inherit From Parent
继承性		Description
简介
clearCSS1IE4+ , NS4+无该属性的值指出了不允许有浮动对象的边。请参阅float属性
floatCSS1IE4+ , NS4+无该属性的值指出了对象是否及如何浮动。请参阅clear属性
clipCSS2IE4+ , NS6+无检索或设置对象的可视区域。区域外的部分是透明的
overflowCSS2IE4+ , NS6+无检索或设置当对象的内容超过其指定高度及宽度时如何管理内容
overflow-xIE专有属性IE4+无检索或设置当对象的内容超过其指定宽度时如何管理内容
overflow-yIE专有属性IE4+无检索或设置当对象的内容超过其指定高度时如何管理内容
displayCSS1/CSS2IE4+ , NS4+无设置或检索对象是否及如何显示
visibilityCSS2IE4+ , NS6+无设置或检索是否显示对象。与display属性不同，此属性为隐藏的对象保留其占据的物理空间

 
 
 
说明：这种颜色是CSS2标准属性。这种颜色是IE建议样式表属性。这种颜色是NS私有属性。这种颜色是目前尚无浏览器支持的属性。
