speak-numeral版本：CSS2　 兼容性：NONE　继承性：有

 
语法：
 
speak-numeral : continuous | digits
 
参数：
 

continuous : 　根据默认语言对数字的词语描述阅读
digits : 　逐位阅读数字
 
说明：
 
设置或检索数字如何发音。
目前IE5.5尚不支持此属性。
对应的脚本特性为speakNumeral。请参阅我编写的其他书目。
 
示例：
 
.telephone { speak-numeral: digits }
