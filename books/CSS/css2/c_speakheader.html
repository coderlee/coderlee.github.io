speak-header版本：CSS2　 兼容性：NONE　继承性：有

 
语法：
 
speak-header : once | always
 
参数：
 
once : 　所有的单元格只有一个表格头
fixed : 　每一组单元格对应一个表头
 
说明：
 
设置或检索表格头与其后的一系列单元格发生多少次关系。
目前IE5.5尚不支持此属性。
对应的脚本特性为speakHeader。请参阅我编写的其他书目。
 
示例：
 
table { speak-header: once }
